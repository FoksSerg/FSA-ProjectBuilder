## История проекта

Проект создан в ноябре 2025 года как универсальный инструмент для модуляризации, сборки и создания дистрибутивов Python проектов.

## Вклад в проект

Основной вклад в разработку:
- Архитектура проекта
- Парсер Python кода (AST)
- Система разборки файлов на модули
- Система сборки модулей в один файл
- Интеграция с PyInstaller, cx_Freeze, Nuitka
- Тестирование и документация
