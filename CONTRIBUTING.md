# Вклад в проект FSA-ProjectBuilder

Спасибо за интерес к проекту! Мы приветствуем любые вклады.

## Как внести вклад

1. Форкните репозиторий
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`)
3. Внесите изменения
4. Закоммитьте изменения (`git commit -m 'Add some AmazingFeature'`)
5. Запушьте в ветку (`git push origin feature/AmazingFeature`)
6. Откройте Pull Request

## Стандарты кода

- Следуйте PEP 8 для стиля кода
- Добавляйте docstrings для всех функций и классов
- Пишите комментарии на русском языке
- Используйте осмысленные имена переменных и функций

## Тестирование

Перед отправкой PR убедитесь, что:
- Все тесты проходят
- Код соответствует стандартам
- Добавлены тесты для новой функциональности

## Вопросы

Если у вас есть вопросы, откройте Issue в репозитории.
