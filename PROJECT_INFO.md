## Как начать разработку

1. Установите зависимости (если нужно)
2. Изучите структуру проекта
3. Начните с реализации core/parser.py
4. Затем реализуйте core/rebuilder.py
5. Затем реализуйте core/builder.py

## Тестирование

Для тестирования самосборки проекта:

```bash
# Разобрать проект на модули
python rebuild.py --project . --file main.py

# Собрать модули обратно
python build.py --project .
```
